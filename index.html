最單純的 Cloudflare Worker + KV

用 Worker 當 API，中間只做兩件事：讀值與寫值。

1. 建立 KV namespace

wrangler kv namespace create AI_CONSULTER_KV

2. 把 id 填回 wrangler.toml

把指令輸出的 id 與 preview_id 填進設定檔。

3. 本機啟動

wrangler dev

4. 寫入 KV

curl -X POST http://127.0.0.1:8787/kv \
  -H "Authorization: Bearer change-me" \
  -H "Content-Type: application/json" \
  -d '{"key":"name","value":"yuanhao"}'

5. 讀取 KV

curl http://127.0.0.1:8787/kv/name \
      -H "Authorization: Bearer change-me"

6. 用 JavaScript 透過 Worker 存取 KV

如果這個頁面不是由同一個 Worker 網址提供，請先填入你的 Worker URL。

const workerUrl = document.querySelector("#worker-url").value.trim() || window.location.origin;
const bearerToken = document.querySelector("#bearer-token").value.trim();
const key = document.querySelector("#kv-key").value.trim();
const value = document.querySelector("#kv-value").value;

async function saveToKv() {
  const response = await fetch(`${workerUrl}/kv`, {
    method: "POST",
    headers: {
      "Authorization": `Bearer ${bearerToken}`,
      "Content-Type": "application/json"
    },
    body: JSON.stringify({ key, value }),
  });

  return response.json();
}

async function loadFromKv() {
  const response = await fetch(`${workerUrl}/kv/${encodeURIComponent(key)}`, {
    headers: {
      "Authorization": `Bearer ${bearerToken}`,
    },
  });
  return response.json();
}

程式碼在同目錄的 worker.js，只有一個 GET 與一個 POST 路由。
